# Computer Vision Bootcamp 

A collection of three computer vision projects progressing from classical image processing to deep learning-based classification.
This was made during my tenure as a Deputy Coordinator for IBOT, the robotics club of IIT Madras and has greatly helped me gain insights and ideas
regarding image processing for robotics.
